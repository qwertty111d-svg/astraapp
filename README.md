# Astra Desktop

Приложение уже привязано к сайту `https://astraboost.ru`.

Что важно:
- API берётся с `https://astraboost.ru`
- Кнопка покупки открывает `https://astraboost.ru/pricing`
- Отдельная локальная админка и локальная база для приложения не нужны

Запуск:
1. Запусти `1_install_dependencies.bat`
2. Запусти `2_run_desktop.bat`
3. Для сборки `.exe` запусти `3_build_windows_exe.bat`

Перед запуском убедись, что сайт уже обновлён и отвечает по адресу:
`https://astraboost.ru/api/desktop/session`

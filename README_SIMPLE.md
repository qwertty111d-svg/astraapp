# Astra Desktop

Что делать:
1. Убедись, что сайт `https://astraboost.ru/api/desktop/session` отвечает.
2. В этой папке запусти `1_install_dependencies.bat`.
3. Потом запусти `2_run_desktop.bat`.
4. Для сборки установщика запусти `3_build_windows_exe.bat`.

Если приложение уже собрано, сайт должен быть обновлён и доступен на домене `astraboost.ru`.
